Django 实战系列
===============

Depot 是《Agile Web Development with Rails》中的一个购物车应用，该教程使许多人走上 Rails 开发的道路。Django 实战系列是 Depot 的 Django 实现，是目前国内 Django 教程里做得最认真也是最好的。该系列用迭代的方法循序渐进地引导初学者，内容很丰富。
